劇団ケッペキは京大公認の演劇サークルです！ 団員は100人ほどで、京都大学以外からも多くの大学の学生が参加しています。 大学一回生でなくても入団できます。また、高校までは未経験という状況で入団した人も沢山います。 年に5～10回公演を行っていますが、決まった時期に公演を必ず行うのではなく、 すべての団員が公演を企画することができるプロデュース制を取っています。 なので劇団固有のカラーにとらわれない、自由なお芝居ができます！

劇団ケッペキでの活動は、きわめて多岐にわたります。役者や脚本家、演出家はもちろんのこと、舞台セットを作ったり、芝居に音楽や照明をのせたり、 衣装を用意したり、公演の宣伝を行ったり、映像を投影したり……と様々なことができます。 発揮できるスキルや身につくスキルは無限大で、あなたの活躍できる場所がここには必ずあります。 あなたも一緒に、お芝居を作りましょう！
